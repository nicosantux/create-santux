# create-santux

Project initializer for React and Next.js projects.

## Usage

npm:

```sh
npm create santux
```

pnpm:

```sh
pnpm create santux
```

bun:

```sh
bun create santux
```

yarn:

```sh
yarn create santux
```

## CLI Options

You can use the flag `-h` or `--help` to see all available options.
